Proyecto GYM Breddy

El sitio web constaba de una pagina ecommerce en el cual el cliente podía acceder a ver los productos existentes y agregarlos a un carrito de compras, pero para esto necesitaba estar registrado, ya que se utilizaba la base de datos para añadir sus productos a su carrito y cuando estuviera listo su carrito podía acceder a realizar un pago con dos opciones una línea de pago y pagar en un Oxxo o por medio de PayPal. Igual se contaba con un apartado para el entrenador de gym pudiera organizar sus horarios y así el cliente pudiera acceder a sus horarios para asignar la hora en la que necesitaba ir.


Desarrollo

  • Se utilizo PHP estructurado
  • Base de datos MySQL
  • Elaboración Manual Técnico
  • Elaboración Manual Usuario
  • Proyecto Finalizado
  • Plugin PayPal

Proyecto ITFreelancer

El sitio web constaba en que el usuario se podía registrar para poder buscar algún profesionista que lo ayudará a desarrollar algún proyecto nuevo o existente, en este caso al registrarse se le mostraba un catálogo de diferentes profesionistas disponibles para atender cualquier proyecto.
Los profesionistas contaban con evaluaciones para que el cliente escogiera de acuerdo a sus necesidades. Una vez que de hizo la selección se le mandaba la solicitud y este decidía si la aceptaba o no; en dado caso de que aceptará se le habilitaba un chat para la comunicación de los términos que se necesitaban y al finalizar todo el proyecto finalizaba el chat para proceder a la evaluación de ambos.


Desarrollo

  • Se utilizo PHP estructurado
  • Base de datos MySQL
  • Elaboración Manual Técnico
  • Elaboración Manual Usuario
  • Proyecto Finalizado

Proyecto GalletasDARM

El sitio web constaba de una página ecommerce en el cual el cliente podía ver los productos existentes y agregarlos a un carrito de compras sin la necesidad de estar registrados en el portal, pero al momento de intentar pagar se te solicitaba iniciar sesión para proceder a realizar el pago de PayPal, al finalizar la compra el cliente puede calificar y agregar sus comentarios al producto, al igual que puede agregar productos a una lista de deseos para comprar después.


Desarrollo

  • Se utilizo PHP MVC
  • Base de datos MySQL
  • Elaboración Manual Técnico
  • Elaboración Manual Usuario
  • Proyecto Finalizado
  • Plugin PayPal
