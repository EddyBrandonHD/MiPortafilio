ITFreelancer

Sitio web para la contratación de profesionistas para la elaboración de proyectos web, en el cual se implementó un chat para la interacción entre cliente y profesionista.

El sitio web constaba en que el usuario se podía registrar para poder buscar algún profesionista que lo ayudará a desarrollar algún proyecto nuevo o existente, en este caso al registrarse se le mostraba un catálogo de diferentes profesionistas disponibles para atender cualquier proyecto. Los profesionistas contaban con evaluaciones para que el cliente escogiera de acuerdo a sus necesidades. Una vez que de hizo la selección se le mandaba la solicitud y este decidía si la aceptaba o no; en dado caso de que aceptará se le habilitaba un chat para la comunicación de los términos que se necesitaban y al finalizar todo el proyecto finalizaba el chat para proceder a la evaluación de ambos.
