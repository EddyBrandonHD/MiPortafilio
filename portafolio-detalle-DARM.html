Galletas DARM

Sitio web ecommerce para la venta de diferentes galletas implementando un carrito de compras y pago PayPal

El sitio web constaba de una página ecommerce en el cual el cliente podía ver los productos existentes y agregarlos a un carrito de compras sin la necesidad de estar registrados en el portal, pero al momento de intentar pagar se te solicitaba iniciar sesión para proceder a realizar el pago de PayPal, al finalizar la compra el cliente puede calificar y agregar sus comentarios al producto, al igual que puede agregar productos a una lista de deseos para comprar después.

Desarrollo

  • Se utilizo PHP MVC
  • Base de datos MySQL
  • Elaboración Manual Técnico
  • Elaboración Manual Usuario
  • Proyecto Finalizado
  • Plugin PayPal

El proyecto no se encuentra en un host actualmente, pero se cuenta con un archivo PDF en el cual se muestra la documentación y capturas de pantalla de una versión preliminar del proyecto. Este proyecto únicamente lo elaboré yo, en donde me encargaba de realizar la documentación y la programación.
